Alquiler de Viviendas

Aplicación web para la gestión de alquileres de viviendas entre propietarios e inquilinos.

Funcionalidades previstas

  • Registro de usuarios
  • Registro de propietarios
  • Alta de propiedades
  • Búsqueda de alojamientos
  • Lista de deseos
  • Reservas inmediatas
  • Solicitudes de reserva
  • Pagos simulados
  • Confirmación o rechazo por parte del propietario
